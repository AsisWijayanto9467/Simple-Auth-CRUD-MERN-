# CRUD Node.js Express MongoDB

Projek Website Dengan fitur Authentikasi serta CRUD dengan metode MERN.

## teknologi yang digunakan
- Node.js
- React
- MongoDB (Mongoose)
- Postman

# Depedensi Node.js
- bcryptjs
- cors
- express
- jsonwebtoken
- mongoose

# Depedensi React
- axios
- bootstrap
- react
- react-router-dom

## Run node.js
node seeder/userSeeder
node server

# Run React
npm run dev
